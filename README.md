# Covid analysis
A small notebook using data from Johns Hopkins CSSE (https://github.com/CSSEGISandData/COVID-19)

## installation
```bash
$ git clone this_repo
$ git submodule init
$ git submodule update
```
Don't forget to update the COVID-19 repo to have latest data:
```bash
$ cd COVID-19
$ git fetch
$ git merge origin/master
```
